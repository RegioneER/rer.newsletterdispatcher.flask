Development
-----------

- Creare un proprio ambiente di sviluppo (venv, virtualenv, whatever...).
- Lanciare il buildout: ``bin/buildout``


- Per avviare ``rq``: ``./bin/rq worker``
- Per avviare ``redis``: ``./bin/redis-server``
- Per avviare l'app ``flask``: ``make dev``
